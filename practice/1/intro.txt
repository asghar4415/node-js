
 1. Create a js file 
     Create a js file with any name. I have created a file with the name server.js. 
     2. Write command npm init -y for creating package.json file 
     Open the terminal and write the command npm init -y. This command will create a package.json file. 
     3. Run npm install express 
     Write the command npm install express. This command will install the express module in your project. 
     4. Create a server.js file 
     Create a server.js file and write the code given below. 
     const express = require('express');

     also create a .gitignore file and add node_modules in it.
